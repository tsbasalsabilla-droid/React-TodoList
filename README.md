# React Todo App 

React Todo App adalah aplikasi Todo List sederhana yang dibuat menggunakan **React Functional Component dan Hooks**.  
Aplikasi ini memiliki tampilan **elegan dengan tema Hitam & Gold**, serta antarmuka modern berbentuk tabel yang mudah digunakan.

---

## Fitur Aplikasi
- Menampilkan daftar todo dalam bentuk tabel
- Menambahkan todo baru
- Menandai todo sebagai selesai (✔)
- Menghapus todo (✕)
- Tampilan modern, rapi, dan elegan
- Data akan hilang saat halaman di-refresh

---

## Teknologi yang Digunakan
- React (Functional Component)
- React Hooks (`useState`)
- Vite
- CSS (Custom Styling)
- JavaScript (ES6)

## Cara Menjalankan Project
1. Clone repository:
```

git clone https://github.com/tsbasalsabilla-droid/React-TodoList
```

2. Masuk ke folder project:
```

cd react-todo-app

```

3. Install dependency:
```

npm install

```

4. Jalankan aplikasi:
```

npm run dev

```

5. Buka browser dan akses:
```

http://localhost:5173

```

---

## Alur Penggunaan Aplikasi

1. Pengguna membuka aplikasi React Todo App melalui browser.
2. Aplikasi menampilkan halaman utama yang berisi:
   - Judul aplikasi
   - Form input untuk menambahkan todo
   - Tabel daftar todo
3. Pengguna memasukkan teks tugas pada kolom input **Tambah todo**.
4. Pengguna menekan tombol **Tambah** untuk menambahkan todo ke dalam daftar.
5. Todo yang ditambahkan akan muncul pada tabel dengan status **Todo**.
6. Pengguna dapat menandai todo sebagai selesai dengan menekan tombol **centang (✔)**.
7. Todo yang telah selesai akan berubah status menjadi **Selesai** dan teks akan dicoret.
8. Pengguna dapat menghapus todo dengan menekan tombol **silang (✕)**.
