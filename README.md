# React Todo App 

React Todo App adalah aplikasi Todo List sederhana yang dibuat menggunakan **React Functional Component dan Hooks**.  
Aplikasi ini memiliki tampilan **elegan dengan tema Hitam & Gold**, serta antarmuka modern berbentuk tabel yang mudah digunakan.

---

## Fitur Aplikasi
- Menampilkan daftar todo dalam bentuk tabel
- Menambahkan todo baru
- Menandai todo sebagai selesai (✔)
- Menghapus todo (✕)
- Tampilan modern, rapi, dan elegan
- Data akan hilang saat halaman di-refresh

---

## Teknologi yang Digunakan
- React (Functional Component)
- React Hooks (`useState`)
- Vite
- CSS (Custom Styling)
- JavaScript (ES6)

---

## ⚙️ Cara Menjalankan Project
1. Clone repository:
```

git clone https://github.com/tsbasalsabilla-droid/React-TodoList
```

2. Masuk ke folder project:
```

cd react-todo-app

```

3. Install dependency:
```

npm install

```

4. Jalankan aplikasi:
```

npm run dev

```

5. Buka browser dan akses:
```

http://localhost:5173

```

---

## Alur Aplikasi
- Todo disimpan menggunakan state React (`useState`)
- Data todo ditampilkan dalam bentuk tabel
- Pengguna dapat:
  - Menambahkan todo melalui form input
  - Menandai todo sebagai selesai dengan tombol centang (✔)
  - Menghapus todo dengan tombol silang (✕)
- Data tidak disimpan ke database atau localStorage, sehingga akan ter-reset saat halaman di-refresh
